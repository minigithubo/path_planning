# Path Planning UI (React)

This is a React-based path planning UI that runs without `npm`.

## How to Run

1. Open `path_planning/frontend/index.html` in your browser (또는 로컬 서버 사용 권장).
2. On the grid:
   - Use `Draw Obstacles` or `Erase Obstacles`
   - Use `Move Start` and `Move Goal`
3. Click `Run Pathfinding` to execute A* path search.

## Features

- Grid-based path planning visualization
- Shortest-path search using the A* algorithm
- Separate rendering for visited nodes and final path
- Adjustable row and column size
- Optional diagonal movement
